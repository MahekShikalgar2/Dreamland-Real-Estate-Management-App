# Dreamland - Real Estate Management App

A **user-friendly Android app** for managing real estate properties, connecting buyers, sellers, and admins seamlessly. Dreamland simplifies property searches, communication, and transactions using **Java, XML, and Firebase**.

---

## üìñ Abstract

Real estate has evolved from ancient land ownership to modern technology-driven platforms. Dreamland addresses the limitations of existing apps (like Magicbricks, 99acres, Housing.com) by offering:

- Accurate property information  
- Direct communication between buyers and sellers  
- Easy property listing and management  
- Secure and efficient property transactions

This app provides a **complete solution for both residential and commercial property management**.

---

## üîπ Features

### Seller
- Upload and manage property details (location, type, size, amenities)  
- Set property price  
- Add high-quality photos and videos  
- Craft appealing property descriptions

### Buyer
- Direct contact with property owners via call or message  
- Personalized property recommendations  
- Pre-book properties of interest  
- Subscription for notifications on new listings

### Admin
- Manage users and verify accounts  
- Approve or remove property listings  
- Monitor financial transactions  
- Provide support and enforce app policies
